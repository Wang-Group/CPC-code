ADTree MATLAB package
- learns ADTree classification models based on datasets
- this package contains several ADTree variants: 
  1. univariate ADTree (classical one)
  2. multivariate ADTree 
  3. sparse ADTree
  4. regularized logistic ADTree (LADTree) 
- there are three folders in this package:
  1. ADTree
  2. Datasets
  3. spasm 
- ADTree folder contains two subfolders:
  1. ADTree_AdaBoost
  2. ADTree_LogitBoost
- Datasets folder contains 20 datasets
- spasm folder contains the Matlab toolbox for sparse statistical modeling from 
  http://www2.imm.dtu.dk/projects/spasm/
